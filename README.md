# Number Guessing Game (C++)

A simple command-line number guessing game built in C++.

## Features
- Random number between 1 and 100
- Difficulty levels (Easy, Medium, Hard)
- Limited attempts based on difficulty
- Input validation
- Win/loss system

## How to Run
1. Compile the program:
   g++ main.cpp -o game

2. Run it:
   ./game
